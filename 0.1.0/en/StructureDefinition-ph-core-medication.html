Resource Profile: PH Core Medication ( Experimental )

Official URL: https://fhir.doh.gov.ph/phcore/StructureDefinition/ph-core-medication Version: 0.1.0
Draft as of 2026-06-17 Computable Name: PHCoreMedication

This resource is primarily used for the identification and definition of a medication, including ingredients, for the purposes of prescribing, dispensing, and administering a medication as well as for making statements about medication use.

Usages:

  • Refer to this Profile: Administered Product, PH Core Medication Administration, PH Core Medication Dispense, PH Core Medication Request... Show 2 more, PH Core Medication Statement and PH Core Procedure
  • Examples for this Profile: Medication/example-medication and Medication/medication-single-example

You can also check for usages in the FHIR IG Statistics
